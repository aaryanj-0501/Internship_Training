# FastAPI MongoDB Notes CRUD API

A simple backend application built using **FastAPI** and **MongoDB** to perform basic **CRUD(Create,Read,Update,Delete)** operations on notes.

# Tech Stack

- **FastAPI** - High performance Python web framework
- **MongoDB** - NoSQL Database
- **Motor** - Async MongoDB Driver
- **Pydantic** - Data Validation

## Features

- Create a Note
- Get all notes
- Get a note by ID
- Update a note
- Delete a note
 
 ## Run and Test the Application

- Run the server using **uvicorn main:app --reload**
- It can be tested locally using http://127.0.0.1:8000/docs
